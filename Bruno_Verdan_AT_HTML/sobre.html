História do seu game:

A ideia veio de muito tempo atrás quando eu já possuía a vontade de colocar os meus amigos em algum trabalho, assim este seria único e mais divertido.

A dificuldade de fazer o jogo em si não foi grande, mas o problema maior foram os bugs. Bug de se ler a mesma carta e considerar como se fossem duas iguais, o contador que não zerava quando se reiniciava o jogo, entre outros.

Integração Interfaces e Javascript

A idéia foi interessante, a integração com as duas matérias. Isso daria espaço para a visualização de como é o mercado, visto que HTML, CSS e Javascript sempre andam juntos.

O grande problema foi a colocação do JS no site, pois já possuía a própria estrutura, que quando foi ajustada, acabou que muitos códigos deram conflitos, por terem o mesmo id e classe. Mas, isso tudo foi superado e, no fim, foi uma boa experiência.

Experiência no desenvolvimento das suas entregas deste bloco

Foi uma boa experiência. Porém, foi uma boa experiência. Eu já tinha algum conhecimento em html e css, assim os primeiros TPs foram muito tranquilos de fazer, mas quando chegou mais para o final, aí complicou um pouco.

E por fim, este AT está sendo muito divertido de se fazer, e ao mesmo tempo trabalhoso. Muitas tags eu nunca tinha mexido e outras até tive que colocar mais atributos que eu não conhecia. Entretanto, apesar de tudo, até agora estou gostando muito de fazer os trabalhos e espero que se mantenha assim até o fim do bloco.
